# Mass Refresher (Chrome/Edge)

Refresh all tabs in all windows with a single click.

## Install (unpacked)
1. Open `chrome://extensions` (or `edge://extensions`).
2. Enable "Developer mode".
3. Click "Load unpacked".
4. Select the `refresher` folder.

## Use
- Click the toolbar icon to refresh every tab in all windows.

## Notes
- Internal browser pages (like `chrome://` or `edge://`) are skipped.
